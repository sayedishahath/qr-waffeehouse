# QR Waffle House - Static Website

A simple, modern static website with 4 navigation links for QR Waffle House.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient background and card-based navigation
- **Interactive Effects**: Hover animations and click ripple effects
- **4 Main Links**:
  - Website
  - Menu
  - Google Review
  - Instagram

## Files

- `index.html` - Main HTML file
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript for interactive functionality

## Customization

To customize the website for your business:

1. **Update URLs**: Edit the URLs in `script.js`:
   - Replace `https://your-website.com` with your actual website
   - Replace `https://your-menu-url.com` with your menu URL
   - Replace the Google Maps URL with your business location
   - Replace the Instagram URL with your Instagram handle

2. **Update Business Name**: Change "QR Waffle House" in `index.html` to your business name

3. **Update Colors**: Modify the gradient colors in `styles.css` to match your brand

## How to Use

1. Open `index.html` in any web browser
2. The website will display 4 clickable cards
3. Each card opens the respective link in a new tab
